# Portfolio Website (GitHub Pages)

## Upload
1) Create a repo named exactly: `yourusername.github.io`
2) Upload **everything** from this folder (keep the same structure)
3) Go to Settings → Pages → Deploy from branch → main / root

## WhatsApp Button
Open `index.html` and replace:
`https://wa.me/971XXXXXXXXX`
with your real number (no +, no spaces), e.g.:
`https://wa.me/971501234567`

## Profile Photo
Replace this file with your own photo (keep the same name):
`assets/images/avatar.png`
